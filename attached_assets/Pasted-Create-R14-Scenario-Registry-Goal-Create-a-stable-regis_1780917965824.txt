Create R14 — Scenario Registry.

Goal:
Create a stable registry of named deterministic scenarios so replay tests, demos, snapshots, and future tools can reference scenarios by ID instead of scattered imports.

Purpose:
Move from ad hoc scenario imports to a central scenario registry that can be safely extended.
