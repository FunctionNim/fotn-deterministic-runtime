# Functions of the Nothing — Continuity Runtime Platform

Functions of the Nothing is no longer primarily a card game.

The card game was the seed architecture.

The current project identity is:

**A deterministic continuity civilization runtime platform.**

The runtime does not simulate animation.
The runtime simulates consequence.

The client does not own truth.
The client renders projected continuity.

---

## Core Doctrine

Pressure → Function → Consequence → Audit → Continuity → Meaning

A player is not merely completing content.
A player is participating in an evolving continuity system where actions are recorded, replayable, attributable, and able to persist into long-term civilization history.

---

## Current Project Stage

The repository is now in:

**Playable continuity embodiment phase.**

The current priority is no longer additional abstraction.
The current priority is proving one playable continuity loop:

Connect → View Board → Select Function Box → Trigger Interaction → See Result → View Replay Frame
